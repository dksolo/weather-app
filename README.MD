# Weather App

## Description

This is a CLI app to call to openweather api. This app is slightly updated version of the PHP Fundamentals tutorial from Laravel.com course. Link is provided below.


## Basic Usage

To use just write "php weather.php" and a name of the city in the terminal like this:

`php weather.php city`

For example, if I want to get the weather in Berlin, I would use the following command : 

`php weather.php Berlin`
